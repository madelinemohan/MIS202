Follow these steps in order to combine several pdf files into one.

  1. Open Adobe Acrobat Pro
  2. From the menu choose File>Combine>Merge Files into a Single PDF...
  3. In the upper left corner of the dialog box select "Add Files.."
  4. Add the .pdf files from wherever they are found...
  5. Make sure to select "Single PDF" in the upper right corner
  6. Click "Combine Files" when ready...
  7. Enter the file name of your choosing, and select a location, then click Save.
